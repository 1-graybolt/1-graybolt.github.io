 # The TCP/IP Model
   
   
 ## Applications

 Application is used to request connection to different part of the internet.

 Protocols in this layers include:

 -HTTPS
 -Email
 -Void
 -SMS
 -IRC
   
 ## Transport

 Transport is used to split data into smaller data and find a connection that has less errors.

 Protocols in this layers include:

 -TCP
 -UDP
   
 ## Internet

 Internet is used to send the packages and make sure that the packages are in the correct place.

 Protocols in this layer include:
 -IP address
 -IPV4
 -IPV6
   
 ## Link (Network Access)

 Link is used to send data between applications over a network.
  
 Protocols in this layers include:
  
 -wifi
 -Bluetooth
 -LTE
 -Ethernet
